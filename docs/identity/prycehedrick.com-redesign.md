# PryceHedrick.com - Enterprise Redesign Spec

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS v4
- Framer Motion

## Design Language
- **Vibe**: Ultra-premium, dark mode, high-contrast, zero clutter. (Inspiration: Stripe, Vercel, Linear).
- **Core Focus**: Enterprise IT Infrastructure & Custom Automation Architect.
- **Constraints**: No blog, no "AI" buzzwords. Strictly professional and targeted.
